# Snake — Professional Edition

A polished, playable Snake game built with `pygame`.

Quick start

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Run the game:

```bash
python snakegame.py
```

Controls

- Arrow keys or WASD: move
- P: pause/unpause
- R: restart after game over
- Q: quit

Files

- `snakegame.py`: game source
- `requirements.txt`: dependencies
- `highscore.txt`: created automatically to store the high score

Enjoy!
